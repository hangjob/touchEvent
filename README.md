# touchEvent
基于Jquery扩展在移动的产生事件，包含，单次触摸事件，两次触摸事件，长按事件，滑屏事件，向上滑动事件，向下滑动事件，向左滑动事件，向右滑动事件

### 事件类型
#### 单次触摸事件
```js
$(el).tap
```

#### 两次触摸事件
```js
$(el).doubleTap
```


#### 长按事件
```js
$(el).longTap
```

#### 滑屏事件
```js
$(el).swipe
```

#### 向上滑动事件
```js
$(el).swipeUp
```

#### 向下滑动事件
```js
$(el).swipeDown
```

#### 向左滑动事件
```js
$(el).swipeLeft
```

#### 向右滑动事件
```js
$(el).swipeRight
```
